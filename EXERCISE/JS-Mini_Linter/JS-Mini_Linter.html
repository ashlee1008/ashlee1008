Mini Linter

In this project, you will use what you know about iterating over arrays to improve the quality of a paragraph and gather some information about that paragraph.

This is the same type of work that word processing software does. Additionally, you may have heard of linting, a process by which text is evaluated and improved by an application. In this project, you will create a miniature version of a linter using array methods that you have learned.

  4. There is an array of words called overusedWords. These are words overused in this story. You want to let the user of your program know how many times they have used these overused words. There are two ways to achieve this. Try it on your own first. If you need help, consult the hint.

    // inputWord 係你想查個隻字
// 依個function用呢check inputWord 在storyWords依個array入面出現左幾多次
// 最後 return 一個 number, 係出現次數
function checkWordNum(inputWord) {
  // reduce 入面要放一個有兩個parameter(參數) 的 function
  // 第一個parameter(參數) 代表 output 同每一次的return value
  // word 代表 array 入面的字
  function countWord(wordNum, word) {
    // 如果 inputWord 係同 word(字會loop over) 一樣
    if (inputWord === word) {
      // wordNum 下一次會加一
      return ++wordNum;
    } else {
      // wordNum 下一次不變
      return wordNum;
    }
  }

  // reduce 除左要function 仲可以入function第1次的value, 依到係 0
  // 0 會係第1次行reduce的 wordNum
  // 最後會 return 最後一個countWord 的 return (wordNum)
  return storyWords.reduce(countWord, 0);
}

    Here are some ideas:

    • For the overused words, remove it every other time it appears.

    • Replaced overused words with something else.

    // 依個function要入3個parameter(參數)
//  words 係你想改的一段字
//  overusedWords 係你想改咩野字
//  replaceWord 可以唔入 入左會replace左你想改的字 唔入會remove左d字
// words 同 overusedWords 的 type 係 array!!
// replaceWord 的 type 係 string!!
function removeAndReplaceOverUsedWord(words, overusedWords, replaceWord) {
  // result會儲低我想要d字 最後會return出去 係依個function的output
  let result = [];

  for (const word of words) {
    // word 係input入呢 每一隻字
    // 如果overusedWords有word會return true
    let isOveruseWord = overusedWords.includes(word);

    // 如果有replaceWord同word係我想改的字, result會加上replaceWord
    // 如果冇replaceWord但係word係我想改的字, result不會加上replaceWord
    // 做到remove overusedWords 的效果
    if (isOveruseWord && replaceWord) {
      result.push(replaceWord);
    }

    // 如果word唔係我想改的字,就直接加入result
    if (!isOveruseWord) {
      result.push(word);
    }
  }
  return result;
}

    // checkWordNum 可以查inputWord在storyWords出現左幾多次
// 詳細睇 4.
function checkWordNum(inputWord) {
  function countWord(wordNum, word) {
    if (inputWord === word) {
      return ++wordNum;
    } else {
      return wordNum;
    }
  }

  return storyWords.reduce(countWord, 0);
}
// Set 兩個variable 用呢儲出現最多次的 字 同 次數
let maxNumOfWord = 0;
let greatestOfWord = "";

// loop over storyWords 一個個字查出現左幾多次
for (let word of storyWords) {
  let wordNum = checkWordNum(word);
  // 如果有出現次數更多的字出現, 會update最多次的 字 同 次數
  if (wordNum > maxNumOfWord) {
    maxNumOfWord = wordNum;
    greatestOfWord = word;
  }
}

// loop完 兩個variable就係出現最多次的 字 同 次數
